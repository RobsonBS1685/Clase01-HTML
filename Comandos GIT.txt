| Utilidad / Acción              | Comando                          | Ejemplo                                                 |
| ------------------------------ | -------------------------------- | ------------------------------------------------------- |
| Configurar nombre de usuario   | `git config --global user.name`  | `git config --global user.name "Eric"`                  |
| Configurar correo              | `git config --global user.email` | `git config --global user.email "correo@gmail.com"`     |
| Ver configuración actual       | `git config --list`              | `git config --list`                                     |
| Inicializar repositorio Git    | `git init`                       | `git init`                                              |
| Clonar un repositorio          | `git clone`                      | `git clone https://github.com/user/proyecto.git`        |
| Ver estado del proyecto        | `git status`                     | `git status`                                            |
| Agregar un archivo al stage    | `git add archivo`                | `git add index.html`                                    |
| Agregar todos los archivos     | `git add .`                      | `git add .`                                             |
| Crear un commit                | `git commit -m`                  | `git commit -m "Agrega login"`                          |
| Ver historial de commits       | `git log`                        | `git log`                                               |
| Ver historial resumido         | `git log --oneline`              | `git log --oneline`                                     |
| Ver diferencias entre cambios  | `git diff`                       | `git diff`                                              |
| Ver ramas existentes           | `git branch`                     | `git branch`                                            |
| Crear una rama                 | `git branch nombre-rama`         | `git branch desarrollo`                                 |
| Cambiar de rama                | `git checkout nombre-rama`       | `git checkout desarrollo`                               |
| Crear y cambiar a una rama     | `git checkout -b`                | `git checkout -b desarrollo`                            |
| Cambiar de rama (nuevo método) | `git switch`                     | `git switch main`                                       |
| Crear rama con switch          | `git switch -c`                  | `git switch -c testing`                                 |
| Fusionar ramas                 | `git merge`                      | `git merge desarrollo`                                  |
| Conectar repositorio remoto    | `git remote add origin`          | `git remote add origin https://github.com/user/app.git` |
| Ver repositorios remotos       | `git remote -v`                  | `git remote -v`                                         |
| Subir cambios a GitHub         | `git push`                       | `git push`                                              |
| Subir primera vez al remoto    | `git push -u origin main`        | `git push -u origin main`                               |
| Descargar cambios del remoto   | `git pull`                       | `git pull`                                              |
| Descargar cambios sin fusionar | `git fetch`                      | `git fetch`                                             |
| Restaurar archivo modificado   | `git restore`                    | `git restore app.js`                                    |
| Sacar archivo del stage        | `git restore --staged`           | `git restore --staged app.js`                           |
| Eliminar rama                  | `git branch -d`                  | `git branch -d testing`                                 |
